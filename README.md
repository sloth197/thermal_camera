# Thermal_Camera with C/C++, Arduino

학교 졸업 프로젝트 | 열화상 카메라를 C/C++과 Arduino로 제어하고, 실시간 온도 데이터를 수집·처리·시각화하는 시스템입니다.  
제조 현장의 온도 모니터링, 설비 상태 감시, 품질 관리 등에 활용 가능한 하드웨어·소프트웨어 통합 시스템입니다.

---

## 📌 프로젝트 개요

- **목표**: 열화상 카메라를 C/C++과 Arduino로 제어하고, 실시간 온도 데이터를 수집·처리·시각화  
- **용도**:  
  - 제조 현장의 온도 모니터링  
  - 설비 상태 감시 (과열 감지)  
  - 품질 관리 (온도 기반 불량 감지)  
- **리스너 연결점**:  
  - Cleber.MC(장비 자동화)와 유사한 “장비 제어 + 센서 데이터 수집”  
  - Cleber.MES(통합생산관리)와 유사한 “실시간 데이터 수집 → 처리 → 관리” 흐름

---

## 🎯 개발 배경

- 제조 현장에서 설비 온도, 공정 온도 등을 실시간으로 모니터링하고, 과열이나 불량을 조기에 감지하는 시스템이 필요  
- 기존 솔루션보다 저렴하고, 제어·데이터 처리를 직접 구현할 수 있는 시스템 개발  
- 하드웨어(열화상 카메라, Arduino)와 소프트웨어(C/C++)를 통합해, 제어·데이터 처리·시각화까지 전 과정을 구현

---

## ⚙️ 시스템 동작 흐름

1. **장비 제어**:  
   - Arduino를 통해 열화상 카메라 제어  
   - 촬영, 온도 측정, 데이터 전송 명령 전달  
2. **데이터 수집**:  
   - 열화상 카메라에서 온도 데이터(픽셀별 온도) 수집  
   - 시리얼 통신 또는 I2C를 통해 PC로 전송  
3. **데이터 처리**:  
   - C/C++ 프로그램에서 온도 데이터를 실시간으로 처리  
   - 최고/최저 온도, 평균 온도, 이상 온도(과열) 감지  
4. **시각화 및 관리**:  
   - 온도 맵(열화상 이미지) 시각화  
   - 이상 온도 알람, 로그 기록, 관리자 대시보드

---

## 🧠 핵심 특징

- **하드웨어·소프트웨어 통합**:  
  - Arduino + 열화상 카메라 + C/C++ 프로그램으로 구성  
  - 장비 제어, 데이터 수집, 처리, 시각화까지 전 과정 구현  
- **실시간 데이터 처리**:  
  - 온도 데이터를 실시간으로 수집·처리  
  - 최고/최저 온도, 평균 온도, 이상 온도 감지  
- **오류 감지 및 알람**:  
  - 설정된 온도 범위를 벗어나면 알람 발생  
  - 로그 기록, 관리자 대시보드에서 상태 확인  
- **제어 시스템 구조**:  
  - 장비 제어 + 센서 데이터 수집 + 실시간 처리 + 시각화  
  - 리스너의 Cleber.MC, Cleber.MES와 유사한 구조

---

## 🛠️ 사용 기술

- **언어**: C, C++  
- **하드웨어**: 열화상 카메라, Arduino, 센서  
- **통신**: 시리얼 통신, I2C  
- **데이터 처리**: 실시간 데이터 처리, 필터링, 통계 계산  
- **시각화**: C/C++ 기반 GUI 또는 간단한 시각화 도구  
- **도구**: Arduino IDE, C/C++ 컴파일러, 시리얼 통신 툴
